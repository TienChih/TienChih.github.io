For any \(a\) and integrable function \(f(t)\) define \(F(x)=\displaystyle \int_a^x f(t)dt\).

Lemma

Given any partition: \(a=t_0, \ldots, t_i, \ldots, t_n=b\) we have that \[\int_a^b f(t)dt = \sum_{i=1}^n [F(t_i)-F(t_{i-1})]\]

Theorem

Suppose \(f\) is integrable.

  1. If \(F(x)=\displaystyle \int_a^x f(t)dt\) then for all \(x_1, x_2\) such that \(a\leq x_1< x_2 \leq b\) then \[\underline{f}(x_2-x_1)\leq F(x_2)-F(x_1) \leq \overline{f}(x_2-x_1).\]

Theorem

Suppose \(f\) is integrable.

  1. If \(F(x)=\displaystyle \int_a^x f(t)dt\) then for all \(x_1, x_2\) such that \(a\leq x_1< x_2 \leq b\) then \[\underline{f}(x_2-x_1)\leq F(x_2)-F(x_1) \leq \overline{f}(x_2-x_1).\]
  2. Converseley, if there was a \(F(x)\) such that for all \(x_1, x_2\) where \(a\leq x_1< x_2 \leq b\) we have \[\underline{f}(x_2-x_1)\leq F(x_2)-F(x_1) \leq \overline{f}(x_2-x_1),\] then \(\displaystyle \int_a^b f(t)dt = F(b)-F(a)\).

Proof of part 2

For \(x_1, x_2\) where \(a\leq x_1< x_2 \leq b\) we have \(\underline{f}(x_2-x_1)\leq F(x_2)-F(x_1) \leq \overline{f}(x_2-x_1),\) then \(\displaystyle \int_a^b f(t)dt = F(b)-F(a)\). As usual: \(a=t_0, \ldots, t_i, \ldots, t_n=b\) is some partition.

\[\underline{f}_1(t_1-t_0) \leq F(t_1)-F(t_0) \leq \overline{f}_1(t_1-t_0)\]

\[\underline{f}_2(t_2-t_1) \leq F(t_2)-F(t_1) \leq \overline{f}_2(t_2-t_1)\]

\[\vdots \]

\[\underline{f}_n(t_n-t_{n-1}) \leq F(t_n)-F(t_{n-1}) \leq \overline{f}_n(t_n-t_{n-1})\]

\[\sum_{i=1}^n\underline{f}_i(t_i-t_{i-1}) \leq \sum_{i=1}^n \left( F(t_i)-F(t_{i-1}) \right) \leq \sum_{i=1}^n\overline{f}_i(t_i-t_{i-1})\]

Proof of part 2

For \(x_1, x_2\) where \(a\leq x_1< x_2 \leq b\) we have \(\underline{f}(x_2-x_1)\leq F(x_2)-F(x_1) \leq \overline{f}(x_2-x_1),\) then \(\displaystyle \int_a^b f(t)dt = F(b)-F(a)\). As usual: \(a=t_0, \ldots, t_i, \ldots, t_n=b\) is some partition.

\[\underline{f}_1(t_1-t_0) \leq F(t_1)-F(t_0) \leq \overline{f}_1(t_1-t_0)\]

\[\underline{f}_2(t_2-t_1) \leq F(t_2)-F(t_1) \leq \overline{f}_2(t_2-t_1)\]

\[\vdots \]

\[\underline{f}_n(t_n-t_{n-1}) \leq F(t_n)-F(t_{n-1}) \leq \overline{f}_n(t_n-t_{n-1})\]

\[\underline{S}\leq \sum_{i=1}^n \left( F(t_i)-F(t_{i-1}) \right) \leq \overline{S}\]

Proof of part 2

For \(x_1, x_2\) where \(a\leq x_1< x_2 \leq b\) we have \(\underline{f}(x_2-x_1)\leq F(x_2)-F(x_1) \leq \overline{f}(x_2-x_1),\) then \(\displaystyle \int_a^b f(t)dt = F(b)-F(a)\). As usual: \(a=t_0, \ldots, t_i, \ldots, t_n=b\) is some partition.

\[\underline{f}_1(t_1-t_0) \leq F(t_1)-F(t_0) \leq \overline{f}_1(t_1-t_0)\]

\[\underline{f}_2(t_2-t_1) \leq F(t_2)-F(t_1) \leq \overline{f}_2(t_2-t_1)\]

\[\vdots \]

\[\underline{f}_n(t_n-t_{n-1}) \leq F(t_n)-F(t_{n-1}) \leq \overline{f}_n(t_n-t_{n-1})\]

\[\underline{S}\leq F(t_n)-F(t_0) \leq \overline{S}\]

Proof of part 2

For \(x_1, x_2\) where \(a\leq x_1< x_2 \leq b\) we have \(\underline{f}(x_2-x_1)\leq F(x_2)-F(x_1) \leq \overline{f}(x_2-x_1),\) then \(\displaystyle \int_a^b f(t)dt = F(b)-F(a)\). As usual: \(a=t_0, \ldots, t_i, \ldots, t_n=b\) is some partition.

\[\underline{f}_1(t_1-t_0) \leq F(t_1)-F(t_0) \leq \overline{f}_1(t_1-t_0)\]

\[\underline{f}_2(t_2-t_1) \leq F(t_2)-F(t_1) \leq \overline{f}_2(t_2-t_1)\]

\[\vdots \]

\[\underline{f}_n(t_n-t_{n-1}) \leq F(t_n)-F(t_{n-1}) \leq \overline{f}_n(t_n-t_{n-1})\]

\[\underline{S}\leq F(b)-F(a) \leq \overline{S}\]

Proof of part 2

\[\underline{S}\leq F(b)-F(a) \leq \overline{S}\]

So \(F(b)-F(a) \) lies bewtween \(\underline{S}, \overline{S}\) for every partion, since \(f\) is integrable, \(I\) exists, so it must equal \(I\), \[\int_a^b f(t)dt=I=F(b)-F(a).\]
