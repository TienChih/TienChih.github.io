As we continue to partition, we always have \(\underline{S}\leq \overline{S}\).

There must be a greatest lower bound and least upper bound. so that \(\underline{S}\leq \leq \underline{S}^* \leq \overline{S}^* \leq \overline{S}\).

If \(\underline{S}^* \neq \overline{S}^*\) then any \(I\) in between can be an "integral" and \(v(t)\) fails to be integrable.

Lemma

If for any \(\epsilon>0\) there is a partition \(a=t_0, \ldots, t_i, \ldots, t_n=b\) so that \(\overline{S}-\underline{S}<\epsilon\) on this partition, then \(v(t)\) is integrable.

If \(v(t)\) is monotonic over \([a,b]\) then it is integrable.
