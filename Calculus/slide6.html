Recall that the rate of change between \(c, t\) of \(f(t)\) is \(\frac{f(t)-f(c)}{t-c}\).

ALSO recall that a function is differentiable at \(c\) if there is a function \(D_cf(t)=\begin{cases}\frac{f(t)-f(c)}{t-c} & t\neq c \\ v & t = c \end{cases}\) such that \(D_cf(t)\) is continuous at \(t=c\).

We say \(v\) is the derivative of \(f\) at \(t=c\), and that \(f\) is differentiable at \(t=c\).

Recall that if \(f(t)=t^2\ , D_cf(t)=\begin{cases}\frac{t^2-c^2}{t-c} & t\neq c \\ v & t = c \end{cases}\).

\(D_cf(t)=\begin{cases}\frac{(t+c)(t-c)}{t-c} & t\neq c \\ v & t = c \end{cases}\).

\(D_cf(t)=\begin{cases} t+c & t\neq c \\ v & t = c \end{cases}\).

\( D_cf(t)=\begin{cases} t+c & t\neq c \\ c + c & t = c \end{cases}\).

\(f'(c)=D_cf(c)=2c\).

\(f'(t)=2t, f'(x)=2x, f'(👹)=2 👹\).

Recall that for \(n\) is a positive integer, \(f(t)=t^n\ , D_cf(t)=\begin{cases}\frac{t^n-c^n}{t-c} & t\neq c \\ v & t = c \end{cases} = \begin{cases}\displaystyle \sum_{i=0}^{n-1}t^ic^{n-1-i} & t\neq c \\ v & t = c \end{cases} \).

Show that \(f'(c)=nc^{n-1}\).

Recall that if \(f(t)=\sqrt{t}\ , D_cf(t)=\begin{cases}\frac{\sqrt{t}-\sqrt{c}}{t-c} & t\neq c \\ k & t = c \end{cases} \).

Show that \(f'(c)=\frac{1}{2\sqrt{c}}\).

Suppose \(f(t)=3\).

Show that \(f'(c)=0\) using this definition.

If \(f(t)\) is differentiable at \(t=c\), then \(f(t)\) is continuous at \(t=c\).

Key: \(f(t) = f(c)+ D_cf(t)\cdot (t-c)\) and \(f(c), t-c, D_cf(t)\) are all functions continuous at \(t=c\).

Power Rule

If \(f(x)=x^n\) then \(f'(x)=nx^{n-1}\).

Constant Rule

If \(f(x)=k\) then \(f'(x)=0\).

Derivative of Circular Functions

If \(f(x)=\sin(x)\) then \(f'(x)=\cos(x)\).

If \(f(x)=\cos(x)\) then \(f'(x)=-\sin(x)\).

Linearity

If \(h(x)=f(x)+g(x)\) then \(h'(x)=f'(x)+g'(x)\).

If \(h(x)=kf(x)\) then \(h'(x)=kf'(x)\).

If \(h(x)=f(x)-g(x)\) then \(h'(x)=f'(x)-g'(x)\).
