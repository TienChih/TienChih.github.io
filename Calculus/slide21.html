Recall: a sequence is a function from a set of integers to the real numbers, usually denoted \(\{a_n\}\).

We say the limit of \(\{a_n\}\) (if it exists) is a number \(L\) so that \(a_n\) is as close to \(L\) as we please, for every \(n\) sufficiently big.

\(a_n=1\) when \(n\) is a power of 2, \(a_n=0\) otherwise. Why doesn't this sequence converge to 0?

Let \(a_n = \frac{1}{n} + (-1)^n \)

To what does \(a_n\) converge?

  1. \(1\)
  2. \(-1\)
  1. Both
  2. Neither

Use the calculator below and adjust \(p\). For which values of \(p\) does \(\frac{1}{n^p}\) converge?

\(\frac{1}{n^p}\to 0\) if \(p>0\).

For which \(r\) does the sequence \(r^n\) converge?

\(r^n\to 0\) if \(-1 < r \leq 1\).

Let \(M\) and \(r\) be positive constants with \(r< 1\) and suppose \(b_n\to K\) for some number \(K\).

  • If \(|a_n-L|\leq M\cdot \frac{1}{n}\) for all sufficiently large \(n\) then \(a_n\to L\).
  • If \(|a_n-L|\leq M\cdot r^n\) for all sufficiently large \(n\) and \(0< r <1\) then \(a_n\to L\).
  • If \(|a_n-L|\leq |b_n-K|\) for all sufficiently large \(n\) then \(a_n\to L\).

If \(\left| \frac{a_{n+1}}{a_n} \right|\leq r <1\) for all sufficiently large \(n\) then \(a_n\to L\).

Let \(\displaystyle \lim_{n\to \infty} a_n = L, \lim_{n\to \infty} b_n = K\).

  • \(\displaystyle \lim_{n\to \infty} (a_n+b_n) = L+K \).
  • \(\displaystyle \lim_{n\to \infty} (a_n-b_n) = L-K \).
  • \(\displaystyle \lim_{n\to \infty} (a_nb_n) = LK \).
  • \(\displaystyle \lim_{n\to \infty} (a_n/b_n) = L/K \) if \(K\neq 0\).
  • \(\displaystyle \lim_{n\to \infty} (a_{n+m}) = L \) for any positive integer \(m\).

If \(f(x)\) is continuous at \(c\) and \(a_n\to c\) then \(f(a_n)\to f(c)\).

  1. Since \(f\) is continuous at \(c, f(x)\) can be made as close to \(f(c)\) by making \(x\) close to \(c\).
  2. Since \(a_n\to c, a_n\) can be made as close to \(c\) as we please by making \(n\) sufficiently big.
  3. So if \(n\) is sufficiently, \(f(a_n)\) can be made as close to \(f(c)\) as we please.

Let \(f(x)\) be a function, defined on some interval \(I\) on all its points except possibly \(c\). If there is a function \(L_cf(x)\) so that:

  1. \(f(x)=L_cf(x)\) for \(x\neq c\).
  2. \(L_cf(x)\) is continuous.

We call \(L_cf(c)\) the limit of \(f\) as \(x\) approaches \(c\) denoted \(\displaystyle \lim_{x\to c}f(x)\).

Note it may be \(L_f(x)\neq f(x)\).

Let \(f(x)\) be a function, defined on some interval \(I\) on all its points except possibly \(c\). If there is a function \(L_cf(x)\) so that:

  1. \(f(x)=L_cf(x)\) for \(x\neq c\).
  2. \(L_cf(x)\) is continuous.

We call \(L_cf(c)\) the limit of \(f\) as \(x\) approaches \(c\) denoted \(\displaystyle \lim_{x\to c}f(x)\).

Note it may be \(L_f(x)\neq f(x)\).

If \(I = (a,c]\) or \([a,c]\) then we call the limit \(\displaystyle \lim_{x\to c^-}f(x)\)

If \(I = [c,b)\) or \([c,b]\) then we call the limit \(\displaystyle \lim_{x\to c^+}f(x)\)

Note it may be \(\displaystyle \lim_{x\to c^-}f(x)\neq \lim_{x\to c^+}f(x)\).

If:

  1. \(f(c)\) is defined.
  2. \(\displaystyle \lim_{x\to c}f(x)\) exists.
  3. \(\displaystyle \lim_{x\to c}f(x)=f(c)\).

Then \(f\) is continuous at \(c\).

Let \(\displaystyle \lim_{n\to \infty} f(x) = L, \lim_{n\to \infty} g(x) = K\).

  • \(\displaystyle \lim_{x\to c} (f(x)+g(x)) = L+K \).
  • \(\displaystyle \lim_{x\to c} (f(x)-g(x)) = L-K \).
  • \(\displaystyle \lim_{x\to c} (f(x)g(x)) = LK \).
  • \(\displaystyle \lim_{x\to c} (f(x)/g(x)) = L/K \) if \(K\neq 0\).

Let \(\displaystyle \lim_{x\to c} f(x) = L, \) then \(\displaystyle \lim_{x\to c}g(f(x)) = \lim_{u\to L} g(u)\) or neither limit exists.

Suppose \(g(x)\) is continuous. If \(|f(x)-L|<|g(x)-g(c)|\) for any \(x\neq c\) in some interval \(I\) containing \(c\) then \(\displaystyle \lim_{x\to c} f(x) = L\).

Discontinuities

If there is a continuous \(g(x)\) so that \(f(x)=g(x), x\neq c\) on some open interval \(I\) containing \(c\) then \(f\) has a removable discontinuity at \(c\).

Hole

\(\displaystyle \lim_{x\to c^-} f(x), \lim_{x\to c^-} f(x) \) are defined, \(\displaystyle \lim_{x\to c} f(x) \) is defined, \(f(c)\) is not defined.

Hole with isolated point

\(\displaystyle \lim_{x\to c^-} f(x), \lim_{x\to c^-} f(x) \) are defined, \(\displaystyle \lim_{x\to c} f(x) \) is defined, \(f(c)\) is defined.

Jump

\(\displaystyle \lim_{x\to c^-} f(x), \lim_{x\to c^-} f(x) \) are defined, \(\displaystyle \lim_{x\to c} f(x) \) is not defined, \(f(c)\) is not defined.

Jump with isolated point

\(\displaystyle \lim_{x\to c^-} f(x), \lim_{x\to c^-} f(x) \) are defined, \(\displaystyle \lim_{x\to c} f(x) \) is not defined, \(f(c)\) is defined.

Assymptote

Either \(\displaystyle \lim_{x\to c^-} f(x), \lim_{x\to c^-} f(x) \) is infinite.
