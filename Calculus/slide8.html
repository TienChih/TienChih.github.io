The "actual" average growth is between the lower and upper bounds.

Let \(a < b\), \(a=t_0 < t_1 < \cdots < t_i < \cdots < t_n=b\). Let \(T_i \in [t_{i-1}, t_i ]\)

We have: \[\underline{v}_i \leq v(T_i) \leq \overline{v}_i \]

Let \(a < b\), \(a=t_0 < t_1 < \cdots < t_i < \cdots < t_n=b\). Let \(T_i \in [t_{i-1}, t_i ]\)

We have: \[\underline{v}_i\cdot (t_{i+1} - t_i) \leq v(T_i)\cdot (t_{i+1} - t_i) \leq \overline{v}_i\cdot (t_{i+1} - t_i) \]

Let \(a < b\), \(a=t_0 < t_1 < \cdots < t_i < \cdots < t_n=b\). Let \(T_i \in [t_{i-1}, t_i ]\)

We have: \[\underline{v}_i\cdot \Delta t_i \leq v(T_i)\cdot \Delta t_i\leq \overline{v}_i\cdot \Delta t_i\]

Let \(a < b\), \(a=t_0 < t_1 < \cdots < t_i < \cdots < t_n=b\). Let \(T_i \in [t_{i-1}, t_i ]\)

We have: \[ \sum_{i=1}^n \underline{v}_i\cdot \Delta t_i\leq \sum_{i=1}^n v(T_i)\cdot \Delta t_i\leq \sum_{i=1}^n \overline{v}_i\cdot \Delta t_i\]

Let \(a < b\), \(a=t_0 < t_1 < \cdots < t_i < \cdots < t_n=b\). Let \(T_i \in [t_{i-1}, t_i ]\)

We have: \[ \sum_{i=1}^n \underline{v}_i\cdot \Delta t_i \leq \sum_{i=1}^n v(T_i) \cdot \Delta t_i \leq \sum_{i=1}^n \overline{v}_i\cdot \Delta t_i\]

Let's focus on say \([t_{i-1}, t_{i}]\). We have a lower bound \(\underline{v}_{i-1}\) and upper bound \(\overline{v}_i\).

Estimates of displacement are now: \[\underline{v}_{i}\cdot (t_i-t_{i-1})\leq \overline{v}_{i} \cdot (t_i-t_{i-1})\]

Let's chop this into 2 bits.

\[\underline{v}_{i}(t_i-t_{*}) +\underline{v}_{i}(t_*-t_{i-1}) \leq \overline{v}_{i}(t_i-t_{*}) + \overline{v}_{i}(t_*-t_{i-1}) \]

Notice: \[\underline{v}_{i}(t_i-t_{*}) +\underline{v}_{i}(t_*-t_{i-1}) \leq \underline{v}_{*}(t_i-t_{*}) + \underline{v}_{i}(t_*-t_{i-1}) \]

Notice: \[\overline{v}_{i}(t_i-t_{*}) +\overline{v}_{*}(t_*-t_{i-1}) \leq \overline{v}_{i}(t_i-t_{*}) + \overline{v}_{i}(t_*-t_{i-1}) \]

So we have improved bound: \[\underline{v}_{*}(t_i-t_{*}) + \underline{v}_{i}(t_*-t_{i-1})\leq \overline{v}_{i}(t_i-t_{*}) +\overline{v}_{*}(t_*-t_{i-1}) \]

Then we can do it again!

A function \(v(t)\) is called integrable over \( [a,b] \) so that given any partition of \([a,b]\) \(a=t_0 < t_1 < \cdots < t_i < \cdots < t_n=b\), there is a unique number \(I\) such that: \[ \underline{S}= \sum_{i=1}^n \underline{v}_i\cdot \Delta t_i\leq I \leq \sum_{i=1}^n \overline{v}_i\cdot \Delta t_i =\overline{S} \]

If such a number exists, we denote it \[I=\int_a^b v(t)dt.\]
