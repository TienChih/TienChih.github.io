The "actual" average growth is between the lower and upper bounds.

Let \(a < b\), \(a=t_0 < t_1 < \cdots < t_i < \cdots < t_n=b\). Let \(T_i \in [t_{i-1}, t_i ]\)

We have: \[\underline{v}_i \leq v(T_i) \leq \overline{v}_i \]

Let \(a < b\), \(a=t_0 < t_1 < \cdots < t_i < \cdots < t_n=b\). Let \(T_i \in [t_{i-1}, t_i ]\)

We have: \[\underline{v}_i(t_{i+1} - t_i) \leq v(T_i)(t_{i+1} - t_i) \leq \overline{v}_i(t_{i+1} - t_i) \]

Let \(a < b\), \(a=t_0 < t_1 < \cdots < t_i < \cdots < t_n=b\). Let \(T_i \in [t_{i-1}, t_i ]\)

We have: \[ \sum_{i=1}^n \underline{v}_i(t_{i+1} - t_i) \leq \sum_{i=1}^n v(T_i)(t_{i+1} - t_i) \leq \sum_{i=1}^n \overline{v}_i(t_{i+1} - t_i) \]

Let's focus on say \([t_{i-1}, t_{i}]\). We have a lower bound \(\underline{v}_{i-1}\) and upper bound \(\overline{v}_i\).

Estimates of displacement are now: \[\underline{v}_{i}(t_i-t_{i-1}) \leq \overline{v}_{i}(t_i-t_{i-1})\]

Let's chop this into 2 bits.

\[\underline{v}_{i}(t_i-t_{*}) +\underline{v}_{i}(t_*-t_{i-1}) \leq \overline{v}_{i}(t_i-t_{*}) + \overline{v}_{i}(t_*-t_{i-1}) \]

Notice: \[\underline{v}_{i}(t_i-t_{*}) +\underline{v}_{i}(t_*-t_{i-1}) \leq \underline{v}_{*}(t_i-t_{i}) + \underline{v}_{i}(t_*-t_{i-1}) \]

Notice: \[\overline{v}_{i}(t_i-t_{*}) +\overline{v}_{*}(t_*-t_{i-1}) \leq \overline{v}_{i}(t_i-t_{*}) + \overline{v}_{i}(t_*-t_{i-1}) \]

So we have improved bound: \[\underline{v}_{*}(t_i-t_{i}) + \underline{v}_{i}(t_*-t_{i-1})\leq \overline{v}_{i}(t_i-t_{*}) +\overline{v}_{*}(t_*-t_{i-1}) \]

Then we can do it again!

A function \(v(t)\) is called integrable over \( [a,b] \) so that given any partition of \([a,b]\) \(a=t_0 < t_1 < \cdots < t_i < \cdots < t_n=b\), there is a unique number \(I\) such that: \[ \sum_{i=1}^n \underline{v}_i(t_i-t_{i-1}) \leq I \leq \sum_{i=1}^n \overline{v}_i(t_i-t_{i-1}) \]

A function \(v(t)\) is called integrable over \( [a,b] \) so that given any partition of \([a,b]\) \(a=t_0 < t_1 < \cdots < t_i < \cdots < t_n=b\), there is a unique number \(I\) such that: \[ \underline{S}= \sum_{i=1}^n \underline{v}_i(t_i-t_{i-1}) \leq I \leq \sum_{i=1}^n \overline{v}_i(t_i-t_{i-1}) =\overline{S} \]

If such a number exists, we denote it \[I=\int_a^b v(t)dt.\]

Yet to be proven facts

If \(v(t)\) is continuous over \([a,b]\) then it is integrable.

Yet to be proven facts

If \(v(t)\) is monotonic over \([a,b]\) then it is integrable.

Yet to be proven facts

If \(v(t)\) is integrable over \([a,b]\) then \(|S-I|\leq |\overline{S}-\underline{S}|\).

Yet to be proven facts

If \(v(t)\) is integrable over \([a,b]\) if and only if \(v(t)\) is integrable over \([a,c]\) and \([c,b]\) for each \(a< c < b \). Moreover \(\displaystyle \int_a^bv(t) dt = \int_a^cv(t)dt + \int_c^bv(t)dt\)
