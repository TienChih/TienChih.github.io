Mean Value Theorem (Integrals)

Let \(f(t)\) be continuous on \([a,b]\). Then there is a \(c, a\leq c\leq b\) where \(f(c)=\displaystyle \frac{1}{b-a}\int_a^b f(t)dt\).

Recall that \[R=\frac{R_1\Delta t_1 + R_2\Delta t_2+ \cdots + R_n\Delta t_n}{\Delta t_1 + \Delta t_2+ \cdots + \Delta t_n}\]

Let \(a=t_0 < t_1 < \cdots < t_n=b\), then: \[R=\frac{R_1(t_1-t_0) + R_2(t_2-t_1)+ \cdots + R_n(t_n-t_{n-1})}{(t_1-t_0) + (t_2-t_1)+ \cdots + (t_n-t_{n-1})}\]

Let \(a=t_0 < t_1 < \cdots < t_n=b\), then: \[R=\frac{R_1(t_1-t_0) + R_2(t_2-t_1)+ \cdots + R_n(t_n-t_{n-1})}{t_n-t_0}\]

Let \(a=t_0 < t_1 < \cdots < t_n=b\), then: \[R=\frac{1}{b-a}\left(R_1(t_1-t_0) + R_2(t_2-t_1)+ \cdots + R_n(t_n-t_{n-1})\right)\]

Let \(a=t_0 < t_1 < \cdots < t_n=b\), then: \[R=\frac{1}{b-a}S\]

The average value of \(f(t)\) over \([a,b]\) is: \[\text{avg}(f)=\frac{1}{b-a}\int_a^bf(t)dt\]

Find the average value of \(f(x)=4x-x^2\) over \([0,4]\).

Find the average value of \(f(x)=\int_1^x \frac{1}{t}dt\) over \([1,x]\).

Find the average value of \(g(t)=\frac{1}{t}\) over \([1,5]\).

Find the average value of \( g(t)=\frac{1}{t}\) over \([1,x]\).

Let \(a(x)=\frac{1}{x-1}\int_1^x \frac{1}{t}dt\). Then \[\frac{1}{x}\leq \frac{1}{x-1}\int_1^x \frac{1}{t}dt \leq 1\]

Let \(\displaystyle a(x)=\frac{1}{x-1}\int_1^x \frac{1}{t}dt\). Then \(\displaystyle a(x^2) = \frac{1}{x^2-1}\int_1^{x^2} \frac{1}{t}dt\)

What is the area of a rectangle with height 1 and base \([1,x]\)?

Let \(\displaystyle a(x)=\frac{1}{x-1}\int_1^x \frac{1}{t}dt\). Then \(\displaystyle a(x^2) = \frac{1}{x^2-1}\int_1^{x^2} \frac{1}{t}dt\)

What is the area of a rectangle with height \(\frac{1}{x}\) and base \([x,x^2]\)?

\[a(x^2) = \frac{1}{x^2-1}\int_1^{x^2} \frac{1}{t}dt< \frac{(x-1)\cdot 1+(x^2-x)\cdot\frac{1}{x}}{x^2-1}\]

\[a(x^2) = \frac{1}{x^2-1}\int_1^{x^2} \frac{1}{t}dt< \frac{(x-1)+(x-1)}{(x-1)(x+1)}\]

\[a(x^2) = \frac{1}{x^2-1}\int_1^{x^2} \frac{1}{t}dt< \frac{2}{x+1}\]

\[a(x) = \frac{1}{x-1}\int_1^{x} \frac{1}{t}dt< \frac{2}{\sqrt{x}+1}\]

\[\frac{\ln(x)}{x} < \frac{1}{x-1}\int_1^{x} \frac{1}{t}dt = \frac{\ln(x)}{x-1}< \frac{2}{\sqrt{x}+1}\]

As \(x\to \infty\), what happens to\(\displaystyle \frac{\ln(x)}{x}\)?

As \(x\to \infty\), and let \(u=e^x\). what happens to \(\displaystyle \frac{\ln(u)}{u} \)?

\[\frac{\ln(x)}{x} < \frac{1}{x-1}\int_1^{x} \frac{1}{t}dt = \frac{\ln(x)}{x-1}< \frac{2}{\sqrt{x}+1}\]

\(\displaystyle\frac{\ln(x)}{x^n} = \frac{1}{n}\frac{\ln(x^n)}{x^n}\) so let \(u=x^n\).

Rewrite \(\displaystyle\frac{1}{n}\frac{\ln(x^n)}{x^n}\) in terms of \(u\) instead of \(x\). What happens as \(x\to \infty\)?

\(\displaystyle\frac{x^n}{e^x} = \left( \frac{x}{e^{x/n}}\right)^n\) so let \(x=n\ln(u)\):

\[\displaystyle\left( \frac{x}{e^{x/n}}\right)^n = \left( \frac{n\ln(u)}{e^{\ln(u)}}\right)^n = \left( \frac{n\ln(u)}{u}\right)^n < \left( \frac{2n}{\sqrt{u}+1}\right)^n = \left( \frac{2n}{\sqrt{e^{x/n}}+1}\right)^n\to 0.\]

From slowest to fastest orders of growth, we have: \[\ln(x) << x^n << e^x\] (where \(n>0\).)
