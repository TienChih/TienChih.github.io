\(Q(t)\) is continuous at \(t=c\) if I can get \(Q(t)\) as close to \(Q(c)\) as I please by moving \(t\) closer to \(c\).

\(Q(x)\) is continuous at \(x=c\) if I can get \(Q(t)\) as close to \(Q(c)\) as I please by moving \(t\) closer to \(c\).

\(Q(x)\) is continuous at \(x=c\) if for any small difference in \(y\) I can pick a small difference in \(x\), so if \(t\) is within the \(x\) distance of \(c\), \(Q(t)\) is within the \(y\) distance of \(Q(c)\)

\(Q(x)\) is continuous at \(x=c\) if for any \(\epsilon>0\) I can pick a small difference in \(x\), so if \(t\) is within the \(x\) distance of \(c\), \(Q(t)\) is within the \(y\) distance of \(Q(c)\)

\(Q(x)\) is continuous at \(x=c\) if for any \(\epsilon>0\) there is a \(\delta>0\), so if \(t\) is within the \(x\) distance of \(c\), \(Q(t)\) is within the \(y\) distance of \(Q(c)\)

\(Q(x)\) is continuous at \(x=c\) if for any \(\epsilon>0\) there is a \(\delta>0\), so if \(|t-c|<\delta\), \(Q(t)\) is within the \(y\) distance of \(Q(c)\)

\(Q(x)\) is continuous at \(x=t\) if for any \(\epsilon>0\) there is a \(\delta>0\), so if \(|t-c|<\delta\), then \(|Q(t)-Q(c)|<\epsilon\).

Let \((a,b)\) be an open interval containing \(c\) in the domain of \(Q\). For any \(t\) in \((a,b)\), we want \(|Q(t)-Q(c)|<\epsilon.\)

For any \(\epsilon>0\), suppose there is a \(M\) so \(|t-c|<\frac{\epsilon}{M}\). (We can think of \(\frac{\epsilon}{M}\) as \(\delta\) .)

If \(|t-c|<\frac{\epsilon}{M}\), then \(M|t-c|<\epsilon\).

If \(|Q(t)-Q(c)|< M|t-c|\), then \(|Q(t)-Q(c)|< M|t-c|<\epsilon\).

Let \((a,b)\) be an open interval containing \(c\) in the domain of \(Q\). For any \(t\) in \((a,b)\), if there is a \(M\) so that \(|Q(t)-Q(c)|< M|t-c|\), then \(Q\) is continuous at \(c\).

If \(Q(t), R(t)\) are continuous on each \(c\) in \((a,b)\), then so is \((Q+R)(t)\).

If \(Q(t), R(t)\) are continuous on each \(c\) in \((a,b)\), then so is \((Q\cdot R)(c)\).

If \(Q(t)\) is continuous at \(c\), and \(R(t)\) is continuous at \(Q(c)\) then \((R\circ Q)(t)\) is continuous at \(c\).

If \(Q(t)\) is continuous at \(c\), then \(kQ(t)\) is continuous at \(c\) for any constant \(k\).

If \(R(t)\) is continuous at \(c\), and \(|Q(t)-Q(c)|<|R(t)-R(c)|\) for every \(t\) in an open interval \((a,b)\) containing \(t\), then \(Q(t)\) is continuous at \(c\).
