Give functions \(f(x), g(x)\) and real number \(a\) such that \(\displaystyle\lim_{x\to a}f(x), \lim_{x\to a}g(x)\) exist:

  • \(\displaystyle \lim_{x\to a}[f(x)+g(x)]=\lim_{x\to a}f(x) + \lim_{x\to a} g(x).\)
  • \(\displaystyle \lim_{x\to a}[c(f(x))]=c\lim_{x\to a}f(x).\)
  • \(\displaystyle \lim_{x\to a}[f(x)g(x)]=\left(\lim_{x\to a}f(x)\right)\left(\lim_{x\to a}g(x)\right).\)
  • \(\displaystyle \lim_{x\to a}[\frac{f(x)}{g(x)}]=\frac{\lim_{x\to a}f(x)}{\lim_{x\to a}g(x)}\) so long as \(\lim_{x\to a} g(x)\neq 0\).

  • Use the first two bullets to show: \(\displaystyle \lim_{x\to a}[f(x)-g(x)]=\lim_{x\to a}f(x) - \lim_{x\to a} g(x).\)
  • Use the third bullet to show: \(\displaystyle \lim_{x\to a}(f(x))^n=\left(\lim_{x\to a}f(x)\right)^n.\)

Consider \[\lim_{h\to 0} \frac{(2+h)^2-2^2}{h}.\]

Consider \[\lim_{h\to 0} \frac{(2+h)^2-2^2}{h}.\]

\[\lim_{h\to 0} \frac{(2+h)^2-2^2}{h} = \lim_{h\to 0}\frac{4+4h+h^2-4}{h}.\]

Consider \[\lim_{h\to 0} \frac{(2+h)^2-2^2}{h}.\]

\[\lim_{h\to 0} \frac{(2+h)^2-2^2}{h} = \lim_{h\to 0}\frac{4h+h^2}{h}.\]

Consider \[\lim_{h\to 0} \frac{(2+h)^2-2^2}{h}.\]

\[\lim_{h\to 0} \frac{(2+h)^2-2^2}{h} = \lim_{h\to 0}\frac{h(4+h)}{h}.\]:

Consider \[\lim_{h\to 0} \frac{(2+h)^2-2^2}{h}.\]

\[\lim_{h\to 0} \frac{(2+h)^2-2^2}{h} = \lim_{h\to 0}\frac{h}{h}(4+h)=\left(\lim_{h\to 0}\frac{h}{h}\right)\left( \lim_{h\to 0} 4+h\right)=1\cdot 4=4.\]

If \(a(x)\leq f(x)\leq b(x) \), what can we say about \(\displaystyle \lim_{x\to c} f(x)\)?

  • Nothing can be concluded.
  • \(\displaystyle \lim_{x\to c} f(x)\leq \lim_{x\to a} b(x)\).
  • \(\displaystyle \lim_{x\to c} a(x)\leq \lim_{x\to a} f(x)\).
  • \(\displaystyle \lim_{x\to c} a(x) \leq \lim_{x\to a} f(x)\leq \lim_{x\to a} b(x)\).

The Squeeze Theorem

If \(a(x)\leq f(x)\leq b(x)\) and \[\lim_{x\to c} a(x)=L=\lim_{x\to c} b(x)\] then \[\lim_{x\to c} f(x)=L.\]
