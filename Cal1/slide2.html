Recall:

Given a function \(f(x)\), the instantaneous rate of change or derivative of \(f(x)\) at \(a\) is the limit as \(h\) approaches 0 of the average rate of change of \(f(x)\) over \([a, a+h]\), or: \[f'(a):=\lim_{h\to 0} \frac{f(a+h)-f(a)}{(a+h)-h}=\lim_{h\to 0} \frac{f(a+h)-f(a)}{h}\]

Given a function \(f(x)\) defined "near" the number \(a\). Then we write \[\lim_{x\to a} f(x)=L\] and say "the limit of \(f(x)\) as \(x\) approaches \(a\) equals \(L\)" if we can get \(f(x)\) as close to \(L\) as we like, by taking \(x\) close to \(a\).

Given a function \(f(x)\) defined "near" the number \(a\). Then we write \[\lim_{x\to a} f(x)=L\] and say "the limit of \(f(x)\) as \(x\) approaches \(a\) equals \(L\)" if for any \(\epsilon>0\) we have \(|f(x)-L|<\epsilon\), by taking \(x\) close to \(a\).

Given a function \(f(x)\) defined "near" the number \(a\). Then we write \[\lim_{x\to a} f(x)=L\] and say "the limit of \(f(x)\) as \(x\) approaches \(a\) equals \(L\)" if for any \(\epsilon>0\) we have \(|f(x)-L|<\epsilon\), whenever \(|x-a|<\delta\), for some \(\delta>0\).

Note: this limit need not exist!

As \(t\) get's closer and closer to 0, what value does \(\frac{\sqrt{t^2+9}-3}{t^2}\) approach?

\[\displaystyle \lim_{t\to 0}\frac{\sqrt{t^2+9}-3}{t^2}=\frac{1}{6}.\]

We use \[\lim_{x\to a^-} f(x)=L^-\] to denote the "limit of \(f(x)\) as \(x\) approaches \(a\) from the left" and \[\lim_{x\to a^+} f(x)=L^+\] to denote the "limit of \(f(x)\) as \(x\) approaches \(a\) from the right".

  • What is \(\displaystyle \lim_{x\to 2^-} f(x)\)?
  • What is \(\displaystyle \lim_{x\to 2^+} f(x)\)?
  • What is \(\displaystyle \lim_{x\to 2} f(x)\)?

\(\lim_{x\to a} f(x)=L\), which exists, if and only if \[\lim_{x\to a^-} f(x)=L=\lim_{x\to a^+} f(x).\]

We say \[\lim_{x\to a} f(x)=\infty,\] if as \(x\) approaches \(a\), \(f(x)\) grows arbitrarily large.
