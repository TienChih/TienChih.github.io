Mixed Partials

We denote \(\frac{\partial }{\partial x_{i_n}}\cdots\frac{\partial }{\partial x_{i_2}}\frac{\partial }{\partial x_{i_1}}[f(\mathbf{x})] \) as \[\frac{\partial^n}{\partial x_{i_n}\cdots \partial x_{i_1}}[f(\mathbf{x})], \frac{\partial^n f}{\partial x_{i_n}\cdots \partial x_{i_1}}, f_{x_{i_n}\cdots x_{i_1}}. \]

So \[\frac{\partial}{\partial y}\left(\frac{\partial}{\partial x}[f(x,y)]\right) = \frac{\partial^2}{\partial y\partial x}[f(x,y)] = \frac{\partial^2 f}{\partial y\partial x}=f_{xy}(x,y).\]

\(C^k\)

Let \(X\) be open in \(\mathbb{R}^n\) and \(f:X\subseteq \mathbb{R}^n\to \mathbb{R}\) so that up to the \(k\)th partial derivatives exist and are continuous is said to be of class \(C^k\).

\(f\) is said to be \(C^\infty\) or smooth if continuous partial derivatives of any order exists.

If \(f:\mathbb{R}^2\to\mathbb{R}\) is at least \(C^2\), then \(f_{xy}=f_{yx}\).

But by starting with \(G(y)=f(a+\Delta x, y)-f(a,y)\), and repeating the same arguments, we have \[\lim_{\Delta x \to 0, \Delta y\to 0} \frac{ D(\Delta x, \Delta y)}{\Delta x \Delta y}=f_{yx}(a,b).\] So \[f_{xy}(a,b)=f_{yx}(a,b).\]

Permutating Derivatives

If \(f:\mathbb{R}^n\to \mathbb{R}\) is \(C^k\) we can perform \(k\) partial derivatives in any order and preserve equality: \[\frac{\partial^k}{\partial x_{i_1}\cdots \partial x_{i_k}}f(\mathbf{x}) = \frac{\partial^k}{\partial x_{j_1}\cdots \partial x_{j_k}}f(\mathbf{x})\]

Consider \(f(x,y)=\begin{cases} \frac{3x^2y-2xy^2}{x^2+y^2} & (x,y)\neq (0,0) \\ 0 & (x,y)=(0,0) \end{cases}\).

Assume \(f_x(0,0)=0, f_y(0,0)=0.\)

  • Use the limit definition of the derivative to compute \[f_{xy}(0,0)=\displaystyle \lim_{h\to 0}\frac{f_x(0,h)-f_x(0,0)}{h}.\]
  • Use the limit definition of the derivative to compute \[f_{yx}(0,0)=\displaystyle \lim_{h\to 0}\frac{f_y(h,0)-f_y(0,0)}{h}.\]
