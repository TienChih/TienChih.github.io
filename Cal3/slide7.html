A partial function

Let \(f:\mathbb{R}^n\to \mathbb{R},\) and let \(\mathbf{x}=(x_1, x_2, \ldots x_n)\) represent a generic point in \(\mathbb{R}^n\). Then the partial function \(F(x_i):\mathbb{R}\to\mathbb{R}\) is a function obtained by fixing all the entries of \(f\) at constants except \(x_i\): \[F(x_i):=f(a_1,\ldots,x_i, \ldots a_n).\]

A partial derivative

The partial derivative of \(f\) with respect to \(x_i\) denotes the single variable derivative of \(F(x_i)\).

Some common notations are: \[\frac{\partial f}{\partial x_i}, D_{x_i}f(x_1, \ldots, x_n), f_{x_i}(x_1, \ldots, x_n).\]

Formally, we have: \[\frac{\partial f}{\partial x_i} = \lim_{h\to 0}\frac{f(x_1, \ldots ,x_i+h,\ldots , x_n) - f(x_ 1, \ldots, x_i,\ldots , x_n) }{h}.\]

Tangent Plane

If the graph of \(z=f(x,y)\) has a tangent plane at \((x_0, y_0, f(x_0,y_0))\), then that tangent plane has equation \[z=f(x_0,y_0)+f_x(x_0,y_0)(x-x_0)+f_y(x_0,y_0)(y-y_0).\]

(Note that \(f_x(x_0, y_0), f_y(x_0,y_0)\) can exist, and a tangent plane may not.)

Differentiability

Let \(X\) be open in \(\mathbb{R}^2\) and \(f(x,y):X\subseteq \mathbb{R}^2\to \mathbb{R}\) so that \(f_x(x_0,y_0), f_y(x_0,y_0)\) exist for some \((x_0,y_0)\).

Then \(f\) is differentiable at \((x_0,y_0)\) if the tangent plane at \((x_0,y_0)\) is a "good" local approximation for \(f\).

Differentiability

Let \(X\) be open in \(\mathbb{R}^2\) and \(f(x,y):X\subseteq \mathbb{R}^2\to \mathbb{R}\) so that \(f_x(x_0,y_0), f_y(x_0,y_0)\) exist for some \((x_0,y_0)\).

Let \[h(x,y)=f(x_0,y_0)+f_x(x_0,y_0)(x-x_0)+f_y(x_0,y_0)(y-y_0).\] Then \(f\) is differentiable at \((x_0,y_0)\) if \[\lim_{(x,y)\to(x_0,y_0)} \frac{f(x,y)-h(x,y)}{\|(x,y)-(x_0,y_0)\|}=0.\]

Differentiability: General

Let \(X\) be open in \(\mathbb{R}^n\) and \(f:X\subseteq \mathbb{R}^n\to \mathbb{R}\) so that \(f_{x_i}(\mathbf{a})\) exists for some \(\mathbf{a}\).

Let \[h(\mathbf{x})=f(\mathbf{a})+\sum_{i=1}^n f_{x_i}(\mathbf{a})(x_i-a_i).\] Then \(f\) is differentiable at \(\mathbf{a}\) if \[\lim_{\mathbf{x}\to \mathbf{a}} \frac{f(\mathbf{x})-h(\mathbf{x})}{\|\mathbf{x}-\mathbf{a}\|}=0.\]

Differentiability: Super General

Let \(X\) be open in \(\mathbb{R}^n\) and \(f:X\subseteq \mathbb{R}^n\to \mathbb{R}^m\) so that \(Df(\mathbf{a})\) exists for some \(\mathbf{a}\).

Let \[h(\mathbf{x})=f(\mathbf{a})+Df(\mathbf{a})(\mathbf{x}-\mathbf{a}).\] Then \(f\) is differentiable at \(\mathbf{a}\) if \[\lim_{\mathbf{x}\to \mathbf{a}} \frac{\|f(\mathbf{x})-h(\mathbf{x})\|}{\|\mathbf{x}-\mathbf{a}\|}=0.\]

Derivative rules still work

Let \(X\) be open in \(\mathbb{R}^n\) and \(f, g:X\subseteq \mathbb{R}^n\to \mathbb{R}^m\) so that \(Df(\mathbf{a}), Dg(\mathbf{a})\) exists for some \(\mathbf{a}\). Let \(k\in \mathbb{R}\).

  • \(D(f+g)(\mathbf{a})=Df(\mathbf{a})+ Dg(\mathbf{a})\)
  • \(D(kf)(\mathbf{a})=kDf(\mathbf{a})\).

If \(m=1\):

  • \(D(fg)(\mathbf{a})=Df(\mathbf{a})\cdot g+ f\cdot Dg(\mathbf{a})\)
  • \(D\left(\frac{f}{g}\right)(\mathbf{a})=\frac{g\cdot Df(\mathbf{a})-f\cdot Dg(\mathbf{a})}{(Dg(\mathbf{a}))^2}\).
