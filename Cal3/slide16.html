Scalar Path Integral

The scalar path integral of a function \(f:\mathbb{R}^n\to\mathbb{R}\) along the path \(C\) parameterized by \(\mathbf{x}\) is \[\int_C f ds= \int_a^b f(\mathbf{x}(t))|\mathbf{x}'(t)|\, dt.\]

Vector Path Integral

The vector path integral of a vector field \(F\) along the path \(C\) parameterized by \(\mathbf{x}\) is \[\int_C F\cdot ds= \int_a^b F(\mathbf{x}(t)) \cdot \mathbf{x}'(t)\, dt = \int_a^b F(\mathbf{x}(t))\cdot \frac{\mathbf{x}'(t)}{|\mathbf{x}'(t)|}|\mathbf{x}'(t)|dt = \int_C F\cdot \mathbf{T} ds .\]

Vector Path Integral

Note for a curve \(C\) and vector field \(F=(F_1, F_2,\ldots, F_n)\) we have: \[\int_C F\, ds= \int_a^b \sum_{i=1}^n F_i(\mathbf{x}(t))\mathbf{x}_i'(t)dt=\int_a^b \sum_{i=1}^n F_i(\mathbf{x}_i)dx_i.\]

So \[\int_C F(x,y,z)\, ds = \int_C F_1(x,y,z)dx + F_2(x,y,z)dy + F_3(x,y,z)dz.\]

Let \(\mathbf{x}:[a,b]\to \mathbb{R}\) be a piecewise \(C^1\) path. Then \(\mathbf{y}:[c,d]\to \mathbb{R}\) is a reparameterization of \(\mathbf{x}\) if there is a bijection \(u:[a,b]\to[c,d]\) such that both \(u, u^{-1}\) are \(C^1\) and \(\mathbf{y}=\mathbf{x}\circ u\).

\(\mathbf{y}\) is orientation preserving if \(u(a)=c, u(b)=d\), and orientation reversing if \(u(a)=d, u(b)=c\).

Let \(\mathbf{x}:[a,b]\to \mathbb{R}\) be a piecewise \(C^1\) path and \(\mathbf{y}:[c,d]\to \mathbb{R}\) be a reparameterization of \(\mathbf{x}\).

Then the vector path integrals \[ \int_c^d F(\mathbf{y})(t)\mathbf{y}'(t)\, dt = \int_a^b F(\mathbf{x})(t)\mathbf{x}'(t)\, dt \] if \(\mathbf{y}\) is orientation preserving, and \[ \int_c^d F(\mathbf{y})(t)\mathbf{y}'(t)\, dt = -\int_a^b F(\mathbf{x})(t)\mathbf{x}'(t)\, dt \] if \(\mathbf{y}\) is orientation reversing.
