Definition of limit, General

We say "the limit of \(f:\mathbb{R}^n\to\mathbb{R}^m\) as \(\mathbf{x}\) approaches \(\mathbf{a}\) is \(\mathbf{L}\) if for any \(\epsilon>0\), there is a \(\delta>0\) so that whenever \(\|\mathbf{x}-\mathbf{a}\|<\delta\) we have \(\|f(\mathbf{x})-\mathbf{L}\|<\epsilon\) "

In Calculus 1, the limit \(L\) exists if the limit approaching \(a\) from the left and the right agree. That is \[\lim_{x\to a^-}f(x)=L=\lim_{x\to a^+} f(x).\]

Limit Algebraic Properties

Let \(F, G:X\subseteq \mathbb{R}^n\to \mathbb{R}^m, f,g,:X\subseteq \mathbb{R}^n\to \mathbb{R}, k\in \mathbb{R}\).

  • \(\displaystyle \lim_{\mathbf{x}\to\mathbf{a} } F(\mathbf{x}) + \lim_{\mathbf{x}\to\mathbf{a} } G(\mathbf{x}) = \lim_{\mathbf{x}\to\mathbf{a} } (F+G)(\mathbf{x})\)
  • \(\displaystyle \lim_{\mathbf{x}\to\mathbf{a} } kF(\mathbf{x}) = k\lim_{\mathbf{x}\to\mathbf{a} } F(\mathbf{x})\)
  • \(\displaystyle \left(\lim_{\mathbf{x}\to\mathbf{a} } f(\mathbf{x})\right)\left(\lim_{\mathbf{x}\to\mathbf{a} } g(\mathbf{x})\right) = \lim_{\mathbf{x}\to\mathbf{a} } (fg)(\mathbf{x})\)
  • \(\displaystyle \frac{\lim_{\mathbf{x}\to\mathbf{a} } f(\mathbf{x})}{\lim_{\mathbf{x}\to\mathbf{a} } g(\mathbf{x})} = k\lim_{\mathbf{x}\to\mathbf{a} } \frac{f}{g}(\mathbf{x})\) so long as \(\lim_{\mathbf{x}\to\mathbf{a} } g(\mathbf{x})\neq 0\).

Consider the function: \[f(x, y)=\frac{3x^3+3xy^2}{x^2+y^2}.\]

Use the algebraic properties of limits to find \[\lim_{(x,y)\to (0,0)}\frac{3x^3+3xy^2}{x^2+y^2}.\] (which exists!)

Continuity

We say \(f:\mathbb{R}^n\to \mathbb{R}^m\) is continuous at \(\mathbf{a}\) if \[\lim_{\mathbf{x}\to\mathbf{a}}f(\mathbf{x})=f(\mathbf{a}).\]

If \(f\) is continuous on it's entire domain, then it is continuous.

Sum, differences products and (when appropriate) quotients of functions continuous at \(\mathbf{a}\) are also continuous at \(\mathbf{a}\).

Consider the functions \[f(x,y)=\begin{cases} \frac{3x^2y+y^3}{x^2+y^2} & (x,y)\neq (0,0) \\ 0 & (x,y)= (0,0) \end{cases}, \]\[g(x,y)=\begin{cases} \frac{-x^2y^2+xy^3}{x^2+y^2} & (x,y)\neq (0,0) \\ 0 & (x,y)= (0,0) \end{cases}.\] Decide if each of them are continuous at \((0,0)\). For either, prove your assertion.
