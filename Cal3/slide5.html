Review of Functions

Given sets \(X, Y\) a function \(f:X\to Y\) is a rule or assignment that associates each element of \(X\) with an element of \(Y\).

If whenever \(x_1\neq x_2\) we have \(f(x_1)\neq f(x_2)\), \(f\) is injective.

The range or image of \(f\) is every element of \(Y\) that is achieved by \(f\): \[\text{Im}(f):=\{y:y=f(x)\ \text{for some $x$ in }X\}.\]

If \(\text{Im}(f)=Y\) we say \(f\) is surjective.

In previous classes, we primarily study functions \(f:\mathbb{R}^1\to\mathbb{R}^1\).

In this class, we primarily study functions \(f:\mathbb{R}^n\to\mathbb{R}^m\).

Consider \(f:\mathbb{R}^2\to\mathbb{R}^1\) defined by \(f(x,y)=\frac{y^3}{12}-y-\frac{x^2}{4}+\frac{7}{2}\). Find \(f(0,0), f(1, -2), f(3,1)\).

Contour plot

Given a function \(f:\mathbb{R}^2\to \mathbb{R}\) the level curve at height \(k\) is the collection of points \((x,y)\in \mathbb{R}^2\) such that \(f(x,y)=k\). \[\{(x,y)\in \mathbb{R}^2 : f(x,y)=k\}.\]

The contour curve at height \(k\) is the collection of points \((x,y, z)\in \mathbb{R}^2\) such that \(f(x,y)=k\) and \(z=k\). \[\{(x,y,z)\in \mathbb{R}^2 : z=f(x,y)=k\}.\]

Given a function \(f:\mathbb{R}^2\to \mathbb{R}\) the level curve is \(\{(x,y)\in \mathbb{R}^2 : f(x,y)=k\}.\) The contour curve is \(\{(x,y,z)\in \mathbb{R}^2 : z=f(x,y)=k\}.\)

Section

Given a function \(f:\mathbb{R}^2\to \mathbb{R}\) the section of the graph of \(f\) by the plane \(x=k\) or \(y=k\) is:\[\{(x,y,z)\in \mathbb{R}^2 : z=f(x,y), x=k\} \text{ or } \{(x,y,z)\in \mathbb{R}^2 : z=f(x,y), y=k\}.\]

Families of surfaces

A sphere centered at \(\mathbf{x}_0(x_0, y_0, z_0)\) with radius \(a\) is the collection of points satisfying \(\|\mathbf{x}-\mathbf{x}_0\|=a\): \[(x-x_0)^2+(y-y_0)^2+(z-z_0)^2=a^2.\]

Families of surfaces

A quadratic surface is a surface described by: \[Ax^2+Bxy+Cxz+DY2+Eyz+Fz^2+Gx+Hy+Iz+J=0\] for real coefficients \(A,B,C,D,E,F,G,H,I,J\).
