Basic (Neutral) Plane Geometry Axioms

A plane is a collection of points and lines (sets of points) satisfying the following axioms:

  1. Each line contains at least two points.
  2. For each line, there is at least one point it does not contain.
  3. There is at least one line.
  4. Given any two points, there is exactly one line containing both.

The Euclidean Axioms

  1. Each line contains at least two points.
  2. For each line \(L\), there is at least one point \(L\) does not contain.
  3. There is at least one line.
  4. Given any two points, there is exactly one line containing both.
  5. Given any line \(L\) and a point \(x\) not on \(L\), there is exactly one line containing \(x\) that doesn't intersect \(L\).
