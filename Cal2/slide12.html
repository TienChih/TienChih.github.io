The Integral Test

Let \(a_n\) be a sequence and let \(f(x)\) be a positive, decreasing, continuous function such that \(a_n=f(n)\).

Then \(\displaystyle \sum_{n=1}^\infty a_n\) converges (diverges) if and only if \(\displaystyle \int_1^\infty f(x)dx\) converges (diverges).

Use the integral test to determine if \(\displaystyle \sum_{n=1}^\infty \frac{1}{n^2+1}\) converges.

Use the integral test to determine if \(\displaystyle \sum_{n=1}^\infty \frac{\ln(n)}{n}\) converges.

For what values of \(p>0\) does \(\displaystyle \sum_{n=1}^\infty \frac{1}{n^p}\) converge?

These are called \(p\)-series.

Proof of the Integral Test

Suppose \(\displaystyle \int_1^\infty f(x)dx\) converges.

\(\displaystyle S_n=\sum_{k=1}^na_k \leq a_1+\int_{1}^{n}f(x)dx.\)

Proof of the Integral Test

Suppose \(\displaystyle \int_1^\infty f(x)dx\) converges.

\(\displaystyle S_n=\sum_{k=1}^n a_k\leq a_1+\int_{1}^{n}f(x)dx\leq \int_{1}^{\infty}f(x)dx.\)
