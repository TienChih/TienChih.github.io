Sums

Given a sequence \(\{a_n\}\), a sequence of partial sums is a sequence of the form \(s_n = \displaystyle \sum_{k=m}^n a_k\) is a sequence defined by \[s_m=\displaystyle \sum_{k=m}^m a_k = a_m, s_{n+1} = \sum_{k=m}^{n+1} a_k := \sum_{k=m}^{n}a_k + a_{n+1}=s_n+a_{n+1}\]

For example, if \[s_n = \displaystyle\sum_{k=1}^n 2k\] then \[s_5 = \sum_{k=1}^5 2k = 2(1) +2(2)+2(3) + 2(4) +2(5) = 2 + 4 + 6+ 8 +10.\]
