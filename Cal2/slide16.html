Ratio Test

Let \(\{a_n\}\) be a sequence such that \( \displaystyle \lim_{n\to \infty} \left| \frac{a_{n+1}}{a_n} \right|=L\).

If \(L < 1\) then \(\sum a_n\) is absolutely convergent.

If \(L > 1\) or \(L=\infty\) then \(\sum a_n\) is divergent.

If \(L =1\)... 🤨

Suppose \(L < 1\), then there is an integer \(M\) and real number \(0< r < 1\) so that for \(n\geq M\), \( \left| \frac{a_{n+1}}{a_n} \right|< r < 1\).

\(|a_{n+1}| < r|a_n| \)

\(|a_{n+2}| < r|a_{n+1}| < r^2|a_{n}| \)

\(|a_{n+3}| < r|a_{n+2}| < r^2|a_{n+1}| < r^3|a_{n}| \)

\(\displaystyle \sum_{n=M}^\infty a_n \leq \sum_{k=0}^\infty (r^k |a_n|) \). Why does \(\sum_{n=M}^\infty a_n\) converge?

Why does \(\sum_{n=1}^\infty a_n\) converge?

Suppose \(L > 1\), then there is an integer \(M\) so that for \(n\geq M\), \( \left| \frac{a_{n+1}}{a_n} \right| > 1\).

Which is greater, \(a_n\) or \(a_{n+1}\)?

Why does \(\sum_{n=1}^\infty a_n\) diverge?

Use the ratio test to see if the following converge: \[\sum_{n=0}^\infty \frac{n^2+5}{2^n}, \sum_{k=1}^\infty \frac{k!}{k^k}, \sum_{n=1}^\infty \frac{3^n}{n^3} \]

Apply the ratio test to the following: \[\sum_{n=1}^\infty \frac{1}{n}, \sum_{n=1}^\infty \frac{1}{n^2} \]

Root Test

Let \(\{a_n\}\) be a sequence such that \( \displaystyle \lim_{n\to \infty} \sqrt[n]{|a_n|}=L\).

If \(L < 1\) then \(\sum a_n\) is absolutely convergent.

If \(L > 1\) or \(L=\infty\) then \(\sum a_n\) is divergent.

If \(L =1\)... 🤨

Use the ratio test to see if the following converge: \[\sum_{n=0}^\infty \left(\frac{2n+3}{3n+2} \right)^n, \sum_{n=1}^\infty \left( \frac{n}{n+1} \right)^n.\]
