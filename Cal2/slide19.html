Polar coordinates are an alternative coordinate system to cartesian coordinates. We express these coordinates as \((r,\theta)\).

The same point can be expressed in multiple ways with polar coordinates.

Find all the polar coordinates of the point \((1, \sqrt{3})\) written with cartesian coordinates.

Find the cartesian coordinates of the point \((-\sqrt{2}, \frac{3\pi}{4})\) written with polar coordinates.

Plot each of the following: \[r=1, r=-1, \theta=\pi/6, \theta=7\pi/6, \theta=-5\pi/6.\]

Plot \[-1\leq r < 1 , 0\leq \theta\leq \pi/2\]

Plot \[-3\leq r \leq 2 , \theta = \pi/4\]

Plot \( r = \sec(\theta)\) (pick a few values for \(\theta\) and try plotting the r values),

Plot \( r = \sin(\theta)\).

Plot \( r = \cos(\theta)+1\).

Plot \( r = \theta\).

Plot \( r = \sin^2(\theta)\).

Plot \( r = 5\sin^2(4\theta)\).

Plot \( r = 5\sin(4\theta)\).

Convert the cartesian values \(x, y, x^2+y^2\) to polar coordinates.
